## 🎯 What's This?

**CoreInventory** is a powerful, web-based inventory management system that combines traditional inventory tracking with AI-powered sales predictions. Built for modern warehouses, it's your smart inventory assistant! 🧠

### ✨ What You Get
- 📊 **Real-time Dashboard** with KPI tracking
- 🧠 **AI Sales Predictions** using LSTM neural networks
- 📈 **Beautiful Analytics** and data visualization
- 🔄 **CSV Data Import** with drag-and-drop interface
- 📱 **Responsive Design** for all devices
- ⚡ **Fast Performance** with optimized backend
- 🎨 **Modern UI/UX** with professional styling
- 🔒 **Robust Error Handling** and validation

## 🚀 Quick Start

```bash
# 1. Clone it
git clone https://github.com/BLACKDEVIL4114/Inventory-Managment-System-.git
cd Inventory-Management-System

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the application!
python run.py
```

**That's it!** 🎉

---

## 🎮 How to Use

### Option 1: Using the Startup Script (Recommended)
```bash
python run.py
# Open http://localhost:5000 in your browser
```
*This script checks dependencies and initializes the database.*

### Option 2: Direct Execution
```bash
python app.py
```
*Directly start the Flask application.*

### Option 3: Windows Batch File
```bash
./start.bat
```
*For quick execution on Windows.*

## 🎨 Features

### 📊 **Dashboard & Analytics**
- **Real-time KPI tracking** with live updates
- **Interactive charts** using Matplotlib and Chart.js
- **Stock level monitoring** with automated alerts
- **Warehouse management** with location tracking
- **Category-wise analysis** for better stock insights

### 🧠 **AI-Powered Predictions**
- **LSTM Neural Networks** for sales forecasting
- **Multi-period predictions** with confidence scores
- **Adaptive training** based on your inventory history
- **Fallback algorithms** for reliable insights

### 🔄 **Data Management**

## 🔧 Technical Highlights

### ✅ **What I Built**
- **Full-stack architecture** with Flask and SQLite
- **AI prediction engine** for forecasting stock needs
- **Real-time dashboard** with live KPI calculations
- **Automated database migrations** using Flask-Migrate
- **Secure Authentication** with Flask-Login and CSRF protection
- **Custom CSS framework** following brand guidelines

### 🧠 **AI Model Architecture**
- **LSTM Layers**: Captures long-term patterns in sales data
- **Dense Layers**: Processes features for final output
- **Dropout & Regularization**: Prevents overfitting
- **Normalization**: Scales data for optimal training stability
