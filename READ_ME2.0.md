# 💎 Inventory Tracker 2.0

A simple, beautiful, and secure way to track your products and warehouse stock.

## 🚀 What is this?
This is a modern **Inventory Management System (IMS)**. It helps businesses keep track of what they have, where it is stored, and when they need to order more. 

## ✨ Key Features
*   **Beautiful Dashboard**: View your business performance with clean charts and easy-to-read cards.
*   **Multi-Warehouse Tracking**: Manage stock across different locations (like Main Warehouse or Store A).
*   **Secure Login**: Only users with a `@gmail.com` email can sign up and log in.
*   **Quick Operations**: Easily add new stock (Receipts) or ship products (Deliveries).
*   **Stock Alerts**: Get clear warnings when a product is running low.

## 🛠️ How to Start the Project

1.  **Install Python**: Make sure you have Python installed on your computer.
2.  **Install Requirements**: Open your terminal and run:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the App**:
    - On Windows: Double-click `start.bat`
    - Or run in terminal: `python app.py`
4.  **Open in Browser**: Go to `http://localhost:5000`

## 👥 User Roles
-   **Managers**: Can add products, create warehouses, and see full analytics.
-   **Staff**: Can view inventory and record daily stock movements.
